订单查询

接口链接：

{接口域名}/order/query

请求方式：

提交方式：POST
Content-Type：application/json

请求参数：

参数名称 是否必填 字段类型 说明
mchid 是 numeric 商户编号
timestamp 是 numeric 下单时间戳（秒），10位数
sign 是 string 签名

返回数据：

参数名称 是否必填 字段类型 说明
code numeric 200：成功；其他：失败
msg string 返回说明
data array 商户信息
timestamp numeric 时间戳（秒），10位数

data数据说明：

参数名称 是否必填 字段类型 说明
mchid string 商户编号
balance string 商户余额
