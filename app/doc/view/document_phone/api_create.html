统一下单

接口链接：

{接口域名}/phone_order/create

请求方式：

提交方式：POST
Content-Type：application/json

请求参数：

参数名称 是否必填 字段类型 说明
mchid 是 numeric 商户编号
out_trade_no 是 string 商户订单号（最长40字母）
phone 是 numeric 手机号
amount 是 numeric 金额
duration 是 numeric 话单时长（分钟：15，30）
notify_url 是 string 回调连接（最长255字母）
timestamp 是 numeric 下单时间戳（秒），10位数
attach 否 string 附加数据（最长255字母）
sign 是 string 签名

返回数据：

参数名称 是否必填 字段类型 说明
code numeric 200：成功；其他：失败
msg string 返回说明
data array 订单信息
timestamp numeric 时间戳（秒），10位数

data数据说明：

参数名称 是否必填 字段类型 说明
order_no string 平台订单号
out_trade_no string 商户订单号
operator numeric 运营商
phone numeric 手机号
amount numeric 充值金额（元）
expire_time numeric 订单过期时间（年-月-日 时:分:秒）
