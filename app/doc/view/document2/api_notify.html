异步通知

规则说明：

1、如果回调成功，期望返回 “success” 字符串。返回其他则表示回调失败。
2、商户收到回调信息，需要进行验证签名，已保证信息安全。

请求方式：

提交方式：POST
Content-Type：application/json

回调参数：

参数名称 是否必填 字段类型 说明
account_name string 账号名称
appid string APPID
data array 订单详情
timestamp numeric 支付时间戳（秒），10位数
sign string 签名

data数据说明：

参数名称 是否必填 字段类型 说明
payer_name string 付款人姓名
payer_phone string 付款人电话
order_no string 订单编号
order_amount numeric 订单金额
order_time datetime 订单时间
order_memo string 订单备注
order_status string 订单状态（如：WAIT_FOR_PAYMENT）
order_status_str string 订单状态说明（如：待支付）
order_cancel_memo string 取消原因
trade_no string 交易单号
