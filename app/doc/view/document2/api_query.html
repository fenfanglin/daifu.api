订单查询

接口链接：

{接口域名}/order/query

请求方式：

提交方式：POST
Content-Type：application/json

请求参数：

参数名称 是否必填 字段类型 说明
mchid 是 numeric 商户编号
out_trade_no 是 string 商户订单号（最长30字母）
timestamp 是 numeric 下单时间戳（秒），10位数
sign 是 string 签名

返回数据：

参数名称 是否必填 字段类型 说明
code numeric 200：成功；其他：失败
msg string 返回说明
data array 订单信息
timestamp numeric 时间戳（秒），10位数

data数据说明：

参数名称 是否必填 字段类型 说明
order_no string 平台订单号
out_trade_no string 商户订单号
pay_amount numeric 实付金额（元）
usdt_amount numeric 实付金额（Usdt），Usdt通道才有
usdt_rate numeric Usdt汇率，Usdt通道才有
usdt_transaction_id numeric Usdt区块链交易id，Usdt通道才有
payer_name string 付款人姓名
attach string 附加数据
success_time numeric 支付时间戳（秒），10位数
notify_time numeric 回调时间戳（秒），10位数
is_notify numeric 1：回调成功；0：未回调/回调失败
status string NOTPAY：未支付；SUCCESS：已支付
